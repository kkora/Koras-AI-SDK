# Support

## Where to get help

1. **Documentation** — start at [docs/index.md](docs/index.md); the
   [troubleshooting guide](docs/troubleshooting/common-errors.md) covers the most common
   errors with resolutions.
2. **GitHub Issues** — bugs and feature requests via the issue templates:
   https://github.com/korastechnologies/koras-ai-sdk/issues
3. **GitHub Discussions** — usage questions and design conversations.

## Before filing a bug

- Reproduce on the latest release.
- Capture the `AiException` details (`Code`, `Provider`, `StatusCode`, `RequestId`) — never
  paste API keys or raw `Authorization` headers.
- Note package versions, TFM, and provider/model.

## Security issues

Never via public issues — see [SECURITY.md](SECURITY.md).

## Commercial support

Not currently offered. Watch ROADMAP.md for changes.
